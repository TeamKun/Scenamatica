### `content` {#message-content}

メッセージの内容を文字列で指定します。  
メッセージの色及び書式は 接頭辞 `§` に続けて[特定の文字](https://minecraft.fandom.com/ja/wiki/Formatting_codes)を指定してください。

:::warning

クリックイベントやホバーイベント等には対応していません。

:::
