## ブロックの破壊 {#break}

<Action name="ブロックの破壊"
        description="ブロックを破壊します。"
        id="block_break"
        args={[
          {
            name: "actor",
            type: ObjectType.STRING,
            anchor: "break-actor",
            description: "ブロックを破壊するアクタの ID です。",
          },
          {
            name: "block",
            type: "Block",
            type_link: "../file-syntax#block-structure",
            description: "破壊するブロックです。",
          },
          {
            name: "dropItems",
            type: ObjectType.BOOLEAN,
            description: "ブロックを破壊した際にアイテムをドロップするかどうかです。"
          }
        ]}
        executable watchable requireable
/>

ブロックの破壊をテストします。

コンディション要求シナリオでこれを使用した場合、指定された場所のブロックの種類が空気ブロックかどうかをテストします。  

:::caution
コンディション要求シナリオでは、以下の引数は使用できません。
+ `actor`
+ `dropItems`
+ [Block 型](../file-syntax#block-structure) の `location` 以外
:::

### `actor` {#break-actor}

アクタは省略可能です。省略した場合は、自然破壊として扱われます。

:::warning

アクタではない通常のプレイヤは指定できません。  
これは、 NMS との相互作用による制限です。

:::
