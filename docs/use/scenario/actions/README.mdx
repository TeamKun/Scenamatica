# アクション

シナリオの有効なアクションの一覧です。

---

## 概要 {#overview}

:::tip

**シナリオやシナリオファイルの概念が初めての方は[こちら](..)を参照して先に理解しておいてください。**

:::

アクションは、シナリオシステムの根幹を成す機能です。  
シナリオで行われる処理の全てはアクションによって実現されます。

## アクションの種類 {#actions}

アクションは膨大な数の種類があるので、各カテゴリ別に分けられています。  
以下のアンカをクリックすると、そのカテゴリのアクション一覧に遷移します。

+ [プレイヤ関連](player)  
  プレイヤの死亡、ゲームモードの変更など
+ [チャット関連](chat)  
  チャットメッセージの送信など
+ [サーバ関連](server)  
  コマンドの送信など
