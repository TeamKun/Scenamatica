### `command` {#command-dispatch-command}

実行するコマンドを指定します。

:::tip

**コマンドの指定には、`/`を付ける必要はありません。**（指定した場合は自動的に除去されます）

:::

### `sender` {#command-dispatch-sender}

コマンドの実行者を指定します。省略した場合は、コンソールが実行者になります。
