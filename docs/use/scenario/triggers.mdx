# トリガ

シナリオの有効なトリガの種類及びその引数について説明しています。

---

## 概要 {#overview}

:::tip

**シナリオやシナリオファイルの概念が初めての方は、先に[こちら](..)を参照して理解しておいてください。**

:::

トリガは、シナリオを実行するための条件を設定するための機能です。

               anchor="trigger"

## トリガの種類 {#triggers}

トリガは、以下の種類があります。

  * [アクション実行 (`on_action`)](#on_action)

### アクション実行 (`on_action`) トリガ {#on_action}

引数に指定されたアクションが実行されたときにシナリオを実行します。  

:::tip

内部的には、[こちら](/docs/use/scenario/elements#scenario-type-action-execution-expect)
のシナリオのアクション実行期待機能を~~使いまわし~~利用しています。

:::

引数は以下のとおりです。

<Object objects={[
  {
    name: "action",
    type: ObjectType.STRING,
    description: "アクションの名前です。",
  },
  {
    name: "with",
    anchor: "action-with",
    type: ObjectType.OBJECT,
    description: "アクションのパラメータです。"
  },
]} />

:::tip

アクションのパラメータは、各アクションのドキュメントを参照してください。

:::
