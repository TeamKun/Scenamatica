## トリガの種類 {#triggers}

トリガは、以下の種類があります。

  * [アクション実行 (`on_action`)](#on_action)

### アクション実行 (`on_action`) トリガ {#on_action}

このトリガは、引数に指定されたアクションが実行されたときにシナリオを実行します。  

:::tip

内部的には、[こちら](/docs/use/scenario/elements#scenario-type-action-execution-expect)のシナリオのアクション実行期待機能と同じです。

:::
