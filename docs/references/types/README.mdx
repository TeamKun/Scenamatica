# 型 - Types

利用できる型について表示しています。

---

:::info

型のリファレンスのページはすべて自動生成されています。  
そのため、型のリファレンスに関する問題がある場合は、[こちら](https://github.com/teamkun/scenamatica)に Issue を作成してください。

:::

## カテゴリ別の型一覧 {#categorised-types}

+ [エンティティ関連](/references/types/entities)  
  エンティティに関する型を提供します。
+ [列挙型関連](/references/types/enums)  
  自動生成された、列挙型の一覧を提供します。
  :::danger
  
  このカテゴリ内のページでは、膨大な数の値が列挙されている可能性があります。  
  そのため、ページの読み込みに時間がかかる場合があります。
  
  :::
+ [インベントリ・アイテム関連](/references/types/inventory_items)  
  インベントリアイテムに関する型を提供します。
+ [その他](/references/types/miscs)  
  その他の型を提供します。
